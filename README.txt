# ProAct

ProAct is a web app made to manage your projects and tasks and see your progress.

## Installation 

To run ProAct locally use Xampp PHP development environment (https://www.apachefriends.org/index.html).

```Run Xampp
Start Apache and MySQL modules.
Press MySQL admin button.
On PHPMyAdmin window insert username 'root' and leave password blank.
Press 'Go' button.
Create a new database by pressing 'New' button.
On 'Databases' window insert database name 'proact' and press 'Create' button.
Press on created 'proact' database.
Press 'Import' button.
Press 'Browse' button, in proact files folder find proact.sql file.
Click 'Open'.
Press 'Go' button.

Navigate to your computer program files.
Go to Xampp -> htdocs.
In Htdocs folder create 'Proact' folder and insert all 'ProAct' files.

Go to your web browser and in web address field type 'localhost/Proact/' and press enter button.
```

## Usage 

```
After login press 'Add project' button and make a project.
To export all projects click 'Export all' icon.
To edit project click 'Edit' icon.
To delete project click 'Delete'icon.
To export project click 'Export' icon.
To add project tasks press on the project in the list and in the project window press 'Add task' button.
To edit project task click 'Edit' icon.
To delete a task click 'Delete' icon.
```
